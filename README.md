 HW4: Perspective

  Creating a scene to be viewed in both orthogonal and perspective projections
  from multiple eye positions directed by user input
  Has 3 view modes:
  1) Overhead Orthogonal
  2) Overhead perspective
  3) First person

  Key bindings:
    arrows     Change view angle
    0          Reset view angle
    ESC        Exit
    (First person)
    w          Forward
    s          Back
    a          Left
    d          Right
